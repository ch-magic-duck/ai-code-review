# 🤖 AI Code Review

AI 代码审查工具，集成 Git 操作和 AI 模型，提供代码审查和通知功能。

## 🚀 当前功能

- 自动提取 Git 差异代码
- 基于 AI 模型进行智能评审
- 评审结果自动提交到指定仓库
- 推送评审结果

## 🛠️ 快速使用

目前只需要在项目内引入 docs 目录下的工作流，修改配置项即可使用
